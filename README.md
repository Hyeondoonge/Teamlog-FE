# Teamlog-Client
공통의 목표를 가진 구성원들이 모여 목표를 위한 과정들을 기록해나가는 그룹단위의 SNS
[teamlog-api-server]()
[push-notification-server](https://github.com/Hyeondoonge/pusher-notification-server)

## Setup
```
npm install
npm start
```

## Technolgies
### product
Netlify, PWA

### framework
React

### api
Google Maps, Pusher

## Features
1. 방문한 장소 map에서 한 눈에 보기
2. 프로젝트 내 포스트 작성
3. 해시태그로 포스트 선별 조회
4. 프로젝트 내 태스크 관리
5. 본인 관련 활동 푸시 알림
